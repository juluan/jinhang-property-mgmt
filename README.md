# 金航建设物业管理系统

基于 Vue 3 + Element Plus + TypeScript 的物业管理平台，包含管理后台与租户小程序端。

## 功能模块

### 出租方管理后台
- 📊 仪表盘 - 核心数据概览
- 🏠 房源管理 - 房源列表、新增房源
- 📋 合同管理 - 合同列表、合同模板
- 💰 费用管理 - 费用配置、账单管理、费用台账
- 🏢 资产管理 - 资产列表、资产统计
- ⚙️ 系统设置 - 用户管理、角色权限、操作日志、通知配置

### 租户端小程序
- 🗺️ 地图找房 - 房源地图浏览与列表筛选
- 📄 合同中心 - 合同模板下载、已签合同上传
- 👤 个人中心 - 账单缴费、合同查看、设置

## 技术栈

- **框架**: Vue 3 (Composition API)
- **UI 组件**: Element Plus
- **语言**: TypeScript
- **构建**: Vite 5
- **状态**: 本地 Mock 数据

## 快速开始

```bash
pnpm install
pnpm dev
```

管理后台: http://localhost:5173/
租户端: http://localhost:5173/tenant
